Sensors Connectivity & Decentralized Sensors Network

Our sensors use the Robonomics firmware, an extended version of the sensor.community firmware, with some sensors added and the data sending scheme changed. The source code can be found at the link.

If you got read-to-use Robonomics board, you can go to the "Connect" section.

Requirements

For Linux:

  1. Add a user to the dialout group (for Ubuntu) to get access to the USB port:

    sudo usermod -a -G dialout $USER
  2. Reboot the computer.

  3. Download Robonomics airrohr-flasher executable from releases.

  4. Change the permissions of the file and execute it:

    chmod +x airrohr-flasher-linux
./airrohr-flasher-linux

For Windows:

  1. Install drivers for USB2serial (in Windows 10 it should start automatically) — Drivers for NodeMCU v3 (CH340): link, alternative link.

  2. Download Robonomics airrohr-flasher executable from releases and run it.

For MacOS:

  1. Install the drivers for USB2serial — Drivers for NodeMCU v3 (CH340): link, alternative link.

  2. Download Robonomics airrohr-flasher executable from releases and run it.

Setup

  1. Connect sensor to PC and run airrohr-flasher program. After opening program you will see the following (depending from you OS):

  2. Board field should automatically fill in; if not, choose the required port form the drop-down list.

    INFO
    If airrohr-flasher can not find your board, make sure you have done the Requirements part properly.

  3. Select the firmware with preferred language, and click Upload. Uploading the firmware will take some time. If you later decide to change language or make clear installation (to reset configuration), go to Erase flash page and press the button to erase the flash memory of sensor.

  4. After downloading the firmware, reboot the ESP (just disconnect and reconnect the USB).

Connect

  1. After rebooting, the board will create a Wi-Fi network named RobonomicsSensor-xxxxxxxxx. Connect to it from your phone or computer: you will see the authorization window (if not, open the browser and go to 192.168.4.1).

  2. Select your Wi-Fi network from the list (or write it yourself if it's not on the list) and fill in the password field.

  3. Write the coordinates of the place, where the sensor will be installed.

    WARNING
    The sensor coordinates will then be displayed on a publicly available map. If you do not want to show your private information, write close, but not exact coordinates.

  4. Click on Save configuration and restart. The board will reboot and connect to the specified Wi-Fi network.

  5. Open Robonomics sensors map and find your place where you installed the sensor. In a couple of minutes you will be able to see your sensor with data on map.

That's all with default installation. For a more advanced setup (connecting additional sensors or sending data to your own server), read the next section.

Additional Configuration

Before configuration, you need to find the address of the sensor in your Wi-Fi network. To do this, you can use airrohr-flasher (your computer must be on the same network as the sensor). Start it and go to the Discovery tab, then press Refresh, wait a moment and your sensor address will appear.

Double-click on this address (or type it into your browser), you will get to the sensor menu:

Now you can start customize your configuration.
